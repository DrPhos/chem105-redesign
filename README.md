# CHEM105 Redesign Project

## Purpose

Redesign the CHEM105 Introductory General, Organic, and Biochemistry for Health Sciences course delivery system.

The initial goal is to replace Vimeo OTT with a self-hosted course website that provides controlled access to lecture videos.

## Phase 1 Goals

- Host lecture videos outside Vimeo OTT
- Provide student access through semester-specific access codes
- Organize videos by chapter and lecture
- Support desktop, tablet, and mobile viewing
- Allow easy semester-to-semester management

## Future Goals

- Canvas integration
- Interactive chemistry tools
- Online practice problems
- Chapter quizzes
- Gradebook integration
- Student progress tracking
- Web versions of existing chemistry apps
